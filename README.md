<h1>API REST Inventario de Productos</h1>
<h2>Endpoints:</h2>
  <p>Obtener toda la lista de Productos: http://localhost:8080/inventario-app/v1/productos <b>(GET)</b></p>
  <p>Obtener informacion especifica de un producto por Id: http://localhost:8080/inventario-app/v1/productos/{id} <b>(GET)</b></p>
  <p>Crear un Producto: http://localhost:8080/inventario-app/v1/productos <b>(POST)</b></p>
  <p>Actualizar un Producto: http://localhost:8080/inventario-app/v1/productos/{id} <b>(PUT)</b></p>
  <p>Eliminar un Producto: http://localhost:8080/inventario-app/v1/productos/{id} <b>(DELETE)</b></p>
